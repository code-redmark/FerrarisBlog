# FerrarisBlog

---

## About FerrarisBlog (English)
FerrarisBlog is a school blog created by students of I.T.I. Galileo Ferraris in Scampia, Naples. The platform collects and shares content from all classes, including news, updates, and articles about school events, projects, and activities. Its goal is to create a collaborative, accessible, and modular website where students and teachers can easily interact with the content.

The project uses Git to practice code management and facilitate collaborative work, and hosts the Front-End on GitHub Pages.

---

## Features to Implement (Current and Future)

### Current Status

**Frontend**
- [x] Main pages: Home, Articles, Contacts
- [ ] Responsive layout for desktop and mobile
- [x] Navigation between sections and categories
- [ ] Display articles with text and images

**Backend**
- [ ] Content managed via local files (e.g., JSON)
- [ ] Adding articles by directly editing files

> The site will remain entirely static at first, as this is a completely new experience and an introduction to web development.

---

### Future Features

**Frontend**
- [ ] Advanced search for articles by keywords or tags
- [ ] Comment section under articles
- [ ] Interface for administrators and classes with different permissions
- [ ] Integration of notifications or external feeds

**Backend**
- [ ] Centralized database for articles, users, and comments (Database as a Service, e.g., Google Firebase)
- [ ] API for reading and writing content
- [ ] Authentication and advanced permission management
- [ ] Automatic data backup and security management

> Of course, the planned objectives are not definitive; the idea was chosen precisely for its scalability, allowing the project to grow as large or as small as needed.

## Cos’è FerrarisBlog (Italiano)
FerrarisBlog è un blog scolastico creato dagli studenti dell'I.T.I. Galileo Ferraris di Scampia, Napoli. La piattaforma raccoglie e condivide contenuti di tutte le classi, incluse notizie, aggiornamenti e articoli su eventi scolastici, progetti e attività didattiche. L’obiettivo è creare un sito collaborativo, accessibile e modulare, dove studenti e insegnanti possano interagire facilmente con i contenuti.

Il progetto utilizza Git per praticare la gestione del codice e facilitare il lavoro collaborativo, oltre all'hosting del Front-End con GitHub Pages.
